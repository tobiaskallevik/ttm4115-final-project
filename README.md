Public repo for code in final delivary of TTM4115 project. Created so that the code can be viewed in an easily accessible way. The STM code are included, but must be ran on their own raspberry PIs. The MTTQ runs on the drone PI. Before running make sure to update the IP addresses to the appropriate once. This applies for the Djano settings file and the state machines. Also make sure to run the init script for a clean backend run.
